## Instalasi
```javascript
1.  git clone https://github.com/Dwi-Merajah/base-pairing
2.  cd /base-pairing
3.  yarn install
4.  node index.js
```
## Penggunaan

1.  input code yang muncul di terminal menggunakan aplikasi WhatsApp di _handphone_ Anda.
2.  Bot akan terhubung dengan akun WhatsApp Anda.

## Catatan
